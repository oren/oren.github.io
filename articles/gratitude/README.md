# Gratitude

## 1. What is it?
Gratitude is a mechanism for scaling relationships.

## 2. The four parts of Gratitude:
1. **Build** - write some code in rust/Go/JS/Typescript
1. **Monetize** - make $$$
1. **Chill** - but don't skip leg day.

## 3. Example

1. Build NVC Chatbot.
1. Charge $299 if customer is happy.
1. Leg day.
