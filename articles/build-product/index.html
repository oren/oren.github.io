How to build a Product That Customers Love

This is a summary of the book Lean Product Playbook.

1. The Process of Building a Product

  1. Create target customer using personas.
  2. Prioritise using the Kano model and Importance vs Satisfaction Framework.
  3. Build mock and test with customers. Your customer gives you better feedback in the solution space.
  4. Pivot if needed.
  5. Build MVP.
  6. Launch your product.
  7. Use Analytics to improve your product.
  8. Use retention rate to measure your product-market fit.
  9. Use cohort analysis to measure product-market fit changes over time.
  10. After you have good retention, improve other micro metrics using AARRR Framework (Acquisition, Activation, Retention, Referal, Revenue).
